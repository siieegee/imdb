How to use cineMatch

Finding your next favourite film takes just a few seconds. Here's everything you need to know.

01
🔍

Search a movie

Type any title into the search bar on the Home page. Results appear instantly as you type — select one to get started.

02
🧬

Explore its cluster

Every movie is assigned to a group based on runtime, release year, vote count, and rating. The feature profile panel shows exactly where your film sits within its group.

03
🎬

Discover similar titles

Up to 12 recommendations appear below, sorted by match score. A same-group bonus rewards films that share the same cinematic DNA.

04
🎛️

Sort & refine

Use the sort buttons to reorder by Best match, Highest rated, Most recent, or Most popular — whatever matters to you.

05
🗂️

Browse by cluster

Not sure where to start? Use the cluster chips below the search bar to browse all movies in a category — Blockbuster Spectacles, Arthouse & Indie, and more.

06
🔁

Go deeper

Click any recommendation card to make it the new selected movie and instantly discover a fresh set of similar titles.

About Us

Behind cineMatch

cineMatch is a movie recommendation system that helps users discover films similar to their favorites. It uses machine learning techniques to analyze movie features such as ratings, runtime, popularity, and release year. By grouping movies into clusters, the system identifies patterns and similarities between titles. This allows users to quickly explore and find new movies that match their preferences.

The Dataset

cineMatch is built on the IMDB public dataset, covering thousands of films across decades. Three tables are combined — title basics, ratings, and name metadata — to form a rich feature matrix for each title.

The Model

Movies are grouped using K-Means clustering (k=5), trained on four normalised features: runtime in minutes, release year, number of votes, and average rating. The model is implemented with scikit-learn and evaluated using Silhouette Score, Davies-Bouldin Index, and Calinski-Harabasz Index.

Group Overview

  • Short cult classics — compact runtime, high cultural footprint
  • Epic prestige dramas — long runtime, high votes & rating
  • Mainstream crowd-pleasers — broad appeal, strong vote counts
  • Arthouse & indie — lower votes, distinctive aesthetic
  • Blockbuster spectacles — modern, long, massively popular
